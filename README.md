# Chocospark
A replica website which is designed to sell 'Chocolates', 'Cakes', 'Pastries' & 'Ice Creams'. The concept of this website is about more constainers & less pages. For coding HTML, CSS, JavaScript & JQuery havebeen used. There are two HTML files (one for Log In Page & the other one is for the other all pages of the website), two CSS files (carring out the designes for two separeate HTML pages) along with two JavaScript files (in which JQuery also have been used frequently) accordingly. All the other codings of  pages available in the website are available on containers in 'index.html' file.
